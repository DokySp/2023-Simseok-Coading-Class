# https://ping1.doky.space API Spec.



## 로그 남기기

### `GET` /

- **query**
  - **name**: 로그 파일 이름
  - **log**: 로그 내용

name에 지정한 로그 파일에다가 log 안에 내용을 `시간 : 내용` 형식으로 붙여줌



### `POST` /

- **body**
  - **name**: 로그 파일 이름
  - **log**: 로그 내용

name에 지정한 로그 파일에다가 log 안에 내용을 `시간 : 내용` 형식으로 붙여줌



## 로그 열람하기

### `GET` /view

- 서버에 저장된 로그 파일 리스트를 보여줌



### `GET` /[파일이름]

- 해당하는 로그 파일의 내용을 보여줌
